## рџљЂ How to Run the App

1. Make sure to open the workspace directory:
   `C:\Users\PC\.gemini\antigravity-ide\scratch\language-learner-app`
2. Create your `.env` file from the template:
   ```bash
   cp .env.example .env
   ```
3. Run the complete application stack (Database + API + Next.js UI) using Docker:
   ```bash
   docker compose up --build
   ```
4. Access the **Frontend UI** at `http://localhost:3000` and the **FastAPI Swagger Docs** at `http://localhost:8000/docs`.
